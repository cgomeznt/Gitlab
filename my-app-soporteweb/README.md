# Multi-Stage con Docker-compose con Nodejs y Angular

Este proyecto muestra como contruir un Multi-Stage, es decir, un docker dentro de un docker 
y luego con la ayuda de otro Multi-Stage correr el contenedor con docker compose

Este proyecto genera un Nodejs y compila un Angular.

En el repo los archivos claves son:
* Dockerfile
* gitlab-ci.yml
* my-docker-compose.yml

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.2.10.
